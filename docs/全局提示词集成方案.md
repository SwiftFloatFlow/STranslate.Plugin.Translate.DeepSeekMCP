# 全局提示词集成实现方案（优化版）

## 需求概述

将主软件提供的全局提示词合并到插件提示词配置的下拉菜单中显示，实现：
1. 全局提示词与局部提示词统一显示在下拉菜单
2. 全局提示词显示时带有"★"前缀标识
3. 局部提示词和全局提示词都使用ID绑定MCP策略
4. 全局提示词不可编辑
5. 支持全局提示词变更实时同步（通过回调机制）

## 显示效果

### 下拉菜单显示
- 局部提示词：`翻译助手`
- 全局提示词：`★全局提示词名称`
- 超长名称自动截断：`★这是一个很长的提...`

### 布局特点
- 下拉框固定宽度 120px
- 超长名称自动截断并显示省略号
- 鼠标悬停显示完整名称（ToolTip）

## ID生成机制

### 统一ID格式

所有提示词（局部和全局）都使用 GUID 生成唯一ID：

```csharp
// ID生成方式（32位无连字符十六进制字符串）
public string Id { get; set; } = Guid.NewGuid().ToString("N");
// 示例：a1b2c3d4e5f67890a1b2c3d4e5f67890
```

### ID唯一性保证

GUID（UUID）天生保证唯一性：
- 理论上 2^128 种可能，几乎不可能重复
- 局部提示词和全局提示词的ID不会冲突
- 无需额外的ID前缀区分

### 提示词来源区分

通过 `Tag` 属性区分来源：
- 局部提示词：`Tag = "Local"` 或空
- 全局提示词：`Tag = "Global:{id}"` 格式

## 接口说明

根据 `参考.txt` 文档，主软件通过 `IPluginContext` 接口暴露全局提示词：

## 设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 显示顺序 | 先局部后全局 | 便于区分，局部提示词使用更频繁 |
| 策略绑定键 | 统一使用ID | ID稳定不变，名称可能被修改 |
| ID生成 | Guid.NewGuid().ToString("N") | 与主程序一致，天生不冲突 |
| 来源标识 | Tag属性 | 使用 `Local`/`Global:{id}` 区分 |
| 全局标识 | ★前缀 | 直观区分，不影响布局 |
| 名称超长 | 截断+省略号 | 保持下拉框宽度稳定 |
| 实时同步 | 回调机制 | 主软件主动通知插件更新 |

## 实时同步机制

### 回调注册

插件初始化时注册全局提示词变更回调：

```csharp
public override void Init(IPluginContext context)
{
    // ... 其他初始化 ...
    
    // 注册全局提示词变更回调（实时通知）
    Context.RegisterGlobalPromptsChangedCallback(OnGlobalPromptsChanged);
}

/// <summary>
/// 全局提示词变更回调处理
/// </summary>
private void OnGlobalPromptsChanged(IReadOnlyList<GlobalPrompt> globalPrompts)
{
    // 在UI线程上执行刷新
    System.Windows.Application.Current.Dispatcher.Invoke(() =>
    {
        RefreshGlobalPromptsFromList(globalPrompts);
    });
}
```

### 回调注销

插件卸载时注销回调，避免内存泄漏：

```csharp
public override void Dispose()
{
    // 注销全局提示词变更回调
    Context?.UnregisterGlobalPromptsChangedCallback(OnGlobalPromptsChanged);
    // ... 其他清理 ...
}
```

## 策略映射键设计

`PromptStrategyMap` 统一使用提示词ID作为键：

```csharp
// 局部提示词策略映射（使用ID）

// 全局提示词策略映射（使用ID）
PromptStrategyMap["f1e2d3c4b5a67890f1e2d3c4b5a67890"] = McpToolStrategy.ToolFirst;
```

## 实现方案

### 1. Settings.cs 修改

#### 1.1 配置版本升级

```csharp
public class Settings
{
    /// <summary>
    /// 配置版本号，用于配置迁移
    /// </summary>
    public int ConfigVersion { get; set; } = 1;
    
    // ... 其他属性 ...
    
    public List<Prompt> Prompts { get; set; } = [];
    
    /// <summary>
    /// 执行配置迁移
    /// </summary>
    public void Migrate()
    {
        if (ConfigVersion < 2)
        {
            // 迁移到版本2：添加策略级设置默认值
            foreach (var strategy in Enum.GetValues<McpToolStrategy>())
            {
                if (!StrategyToolChainDisplay.ContainsKey(strategy))
                    StrategyToolChainDisplay[strategy] = false;
                if (!StrategyToolResultDisplayModes.ContainsKey(strategy))
                    StrategyToolResultDisplayModes[strategy] = ToolResultDisplayMode.Disabled;
                if (!StrategyConsecutiveToolLimits.ContainsKey(strategy))
                    StrategyConsecutiveToolLimits[strategy] = StrategyConsecutiveLimitHelper.DEFAULT_LIMIT;
                if (!StrategyTotalToolCallsLimits.ContainsKey(strategy))
                    StrategyTotalToolCallsLimits[strategy] = StrategyTotalToolCallsHelper.DEFAULT_LIMIT;
            }
            ConfigVersion = 2;
        }
        
        // 版本3：提示词策略映射迁移（名称 -> ID）
        if (ConfigVersion < 3)
        {
            MigratePromptStrategyMapToIdBased();
            ConfigVersion = 3;
        }
    }
    
    /// <summary>
    /// 迁移提示词策略映射：从名称绑定迁移到ID绑定
    /// </summary>
    private void MigratePromptStrategyMapToIdBased()
    {
        var newMap = new Dictionary<string, McpToolStrategy>();
        
        foreach (var prompt in Prompts)
        {
            // 确保每个提示词都有ID
            if (string.IsNullOrEmpty(prompt.Id))
            {
                prompt.Id = Guid.NewGuid().ToString("N");
            }
            
            // 设置来源标识
            prompt.Tag ??= "Local";
            
            // 迁移策略映射
            if (PromptStrategyMap.TryGetValue(prompt.Name, out var strategy))
            {
                newMap[prompt.Id] = strategy;
            }
        }
        
        PromptStrategyMap = newMap;
    }
}
```

```csharp
/// <summary>
/// 合并后的所有提示词（局部 + 全局）

public override ObservableCollection<Prompt> Prompts { get; } = new();

    var tag = prompt.Tag?.ToString();
    return tag?.StartsWith("Global:") == true;

/// <summary>

/// </summary>

{
    if (IsGlobalPrompt(prompt))
    {
        return $"{prompt.Name} [全局]";
    }
    return prompt.Name;
```

#### 2.2 修改 Init 方法

```csharp
public override void Init(IPluginContext context)
{
    Context = context;
    Settings = context.LoadSettingStorage<Settings>();
    
    // 执行配置迁移（包括名称->ID的策略迁移）
    Settings.Migrate();
    
    // 加载局部提示词
    foreach (var prompt in Settings.Prompts)
    {
        // 确保每个提示词都有ID（兼容旧配置）
        if (string.IsNullOrEmpty(prompt.Id))
        {
            prompt.Id = Guid.NewGuid().ToString("N");
        }
        prompt.Tag ??= "Local";
        Prompts.Add(prompt);
    }
    
    // 加载全局提示词
    LoadGlobalPrompts();
    
    // 初始化MCP客户端...
    if (Settings.EnableMcp && Settings.McpServers.Count > 0)
    {
        InitializeMcpClients();
    }
}

/// <summary>
/// 加载全局提示词并合并到 Prompts 列表
/// </summary>
private void LoadGlobalPrompts()
{
    try
    {
        var globalPrompts = Context.GetGlobalPrompts();
        
        foreach (var globalPrompt in globalPrompts)
        {
            var prompt = globalPrompt.ToPrompt(true);
            prompt.Tag = $"Global:{globalPrompt.Id}";  // 标记来源
            Prompts.Add(prompt);
        }
        
        if (ShouldLog(1) && globalPrompts.Count > 0)
            Context.Logger.LogInformation("[提示词] 已加载 {Count} 个全局提示词", globalPrompts.Count);
    }
    catch (Exception ex)
    {
        Context.Logger.LogError(ex, "[提示词] 加载全局提示词失败");
    }
}
```

#### 2.3 修改 SelectPrompt 方法

```csharp
public override void SelectPrompt(Prompt? prompt)

    base.SelectPrompt(prompt);
```

```csharp
private void UpdateSelectedPromptStrategy()
{
    if (Main.SelectedPrompt == null)

        SelectedPromptStrategy = PromptStrategyOptions.First();
```

```csharp
private void SavePromptStrategy()
{
    if (Main.SelectedPrompt == null || SelectedPromptStrategy == null)
        return;

    // 统一使用ID作为键
    var strategyKey = Main.GetStrategyKey(Main.SelectedPrompt);

    _context.SaveSettingStorage<Settings>();
    
    var promptName = Main.SelectedPrompt.Name;
    _context.Logger.LogInformation(
        $"[提示词策略] 提示词 '{promptName}'(ID:{strategyKey[..8]}...) 的策略已设置为: {PromptStrategyHelper.GetStrategyName(newStrategy)}");
```

```csharp
[RelayCommand]
private void EditPrompt()
{
    // 全局提示词不可编辑
    if (Main.SelectedPrompt != null && Main.IsGlobalPrompt(Main.SelectedPrompt))
    {
        _context.Logger.LogWarning("[提示词] 全局提示词不可编辑");
        return;
    }
    
    var dialog = _context.GetPromptEditWindow(Main.Prompts);

    if (dialog.ShowDialog() == true)
    {
        // 只保存局部提示词
        var localPrompts = Main.Prompts
            .Where(p => !Main.IsGlobalPrompt(p))
            .Select(p => p.Clone())
            .ToList();
        _settings.Prompts = localPrompts;
        _context.SaveSettingStorage<Settings>();
        Main.SelectedPrompt = Main.Prompts.FirstOrDefault(p => p.IsEnabled);
    }
}
```

### 5. 新增转换器

#### 5.1 PromptDisplayNameConverter.cs

```csharp
using System.Globalization;
using System.Windows.Data;
using STranslate.Plugin;

namespace STranslate.Plugin.Translate.DeepSeek.Converters;

/// <summary>
/// 提示词显示名称转换器：为全局提示词添加"★"前缀标识，并支持超长名称截断
/// </summary>
public class PromptDisplayNameConverter : IValueConverter
{
    public const int MaxDisplayLength = 15;
    public const string Ellipsis = "...";
    public const string GlobalPrefix = "★";

    public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
    {
        if (value is Prompt prompt)
        {
            return GetDisplayName(prompt);
        }
        return value?.ToString() ?? "";
    }

    public static string GetDisplayName(Prompt? prompt)
    {
        if (prompt == null) return "";
        
        var tag = prompt.Tag?.ToString();
        bool isGlobal = tag?.StartsWith("Global:") == true;
        string prefix = isGlobal ? GlobalPrefix : "";
        string name = prompt.Name ?? "";
        
        // 如果超长，截断名称（保留前缀）
        if (prefix.Length + name.Length > MaxDisplayLength)
        {
            int availableLength = MaxDisplayLength - prefix.Length - Ellipsis.Length;
            if (availableLength > 0)
            {
                name = name.Substring(0, availableLength) + Ellipsis;
            }
            else
            {
                name = Ellipsis;
            }
        }
        
        return prefix + name;
    }

    public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
    {
        throw new NotImplementedException();
    }
}
```

**转换器特点**：
- 全局提示词添加 `★` 前缀
- 局部提示词不添加任何标识
- 超过15个字符自动截断显示省略号
- 鼠标悬停显示完整名称（通过ToolTip）

## 注意事项

1. **ID唯一性**：GUID天生唯一，无需担心局部与全局冲突
2. **迁移时机**：配置迁移在 `Init` 方法中自动执行

4. **策略键统一**：局部和全局都使用ID，代码简洁一致
